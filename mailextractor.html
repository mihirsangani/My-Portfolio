Mail Extractor

  1. Home
  2. Mail Extractor

Project information

  • Category : Web Development
  • Project URL : https://mailextractor.io
  • Technology Used : WordPress | ReactJS | NodeJS | PostgreSQL

Project description

A cloud-based software that automatically reads emails, extracts important data, and sends them to applications like Trello, Slack, Google Sheets, etc. using user-defined custom parsing rules.

  • Create custom parsing rules using a a step-by-step wizard that graphically guides you through the process of splitting your email content and extracting key data.
  • Transfer extracted data to Google Spreadsheet , Trello & Slack automatically using our inbuilt integration or to other applications using Webhooks.
  • Software for processing Emails all at once. Compatible with every email provider. Integrates well with major email providers like Gmail and Outlook.

Third party APIs used

  • Sign-In / Sign-Up : Used Google Workspace APIs for sign-in & sign-up.
  • Slack API : Integrated Slack API to send reports to users.
  • Google Storage Bucket : Used Google Storage Bucket to upload and download Images, Videos, Documents, etc...
  • Node Schedule : Used node-schedule library to schedule jobs for execution at specific dates like send reminders, remove data from database, etc...
  • Google Console Cloud : Used google OAuth 2.0 library for sending mail to users using Gmail API.
